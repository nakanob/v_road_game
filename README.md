# Road Trip Drive v8

## 主な修正
- 横断歩道の白線を道路進行方向と平行に修正
- 白いエリア切替ゲートを削除
- キャンピングカーをトヨタ・ダイナ系トラックベースのキャブコン形状へ再設計
- 水平な屋根、天窓なし、前面・運転席・助手席窓、左側中央扉、後面窓、屋根内に収まるはしごを反映
- ヘッドライトの照射角度を下げて路面照射を強化
- ダスト表現を停止
- 道路外へ向かったときは横滑りせず減速
- 一軒家の壁・屋根・扉・窓位置をランダム化
- 前進時の左右操作を修正
- スマートフォン用バーチャルパッドを追加
- トンネルを自然な岩山型へ変更
- 川・橋の上下位置とサイズを再調整
- 木・草を軽量なインスタンシング中心で大幅増量
- ゴールに20人、バーベキュー、照明、歓迎演出を追加
