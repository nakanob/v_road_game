# Road Trip Drive - Lightweight Camper Edition

## 主な変更
- 外部GLBモデルを完全に廃止し、軽量な白いキャンピングカーをThree.jsの基本形状だけで構成
- 連続したカーブ座標を使用し、車両位置のカクつきを解消
- カメラは道路方向と位置だけを滑らかに追従し、車体の傾きには連動しない
- 左右キーの方向を修正
- 建物の外壁・窓・入口、樹皮・枝・葉、草、岩、土を軽量なCanvasTextureで強化
- キャンプ場の夜空に固定の星を表示
- エリア切り替えを約3秒かけて自然に補間
- ゴール後は車両を停止し、カメラ操作を継続可能

## 操作
- W / ↑: 加速
- S / ↓: ブレーキ・後退
- A / ←: 左
- D / →: 右
- マウスドラッグ: カメラ回転
- ホイール: ズーム
- C: カメラ初期化
